void cCommand ( List * clients)

Executes the 'c' command.

Executes the 'c' command, listing all invoices for a client or all invoices in chronological order if no argument is provided.

Parameters
clientsPointer to the list of clients in the system.
  1. Command invoked without arguments: List all system invoices
  1. Specific client name provided: Parse and fetch their records
