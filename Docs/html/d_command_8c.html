Executes the 'd' command, deleting invoices or removing unused products. Depending on the parsed arguments, either deletes an invoice from global memory and client history, or removes a specified quantity of available product stock.

Executes the 'd' command, searching for all invoices containing a specific product. Expects an EAN code from standard input.

Parameters
productsPointer to the global list of system products.
invoicesPointer to the global list of system invoices.
systemContextPointer to the global system context to update stats.

1 Arg provided: Assume it is an invoice ID for deletion

2 Args provided: Parse EAN and units to deduct product stock
