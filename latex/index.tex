\chapter{IAED 25/\+26 -\/ Invoicing System }
\hypertarget{index}{}\label{index}\index{IAED 25/26 -\/ Invoicing System@{IAED 25/26 -\/ Invoicing System}}
This project implements an invoicing management system for a store, allowing efficient management of products, clients, and invoices. It utilizes hybrid data structures (Doubly Linked Lists and AVL Trees) to ensure O(log N) search performance and O(1) chronological insertions.

Main features:\+
\begin{DoxyItemize}
\item \doxylink{struct_product}{Product} stock management (EAN-\/8/\+13).
\item Processing of temporary shopping baskets.
\item Issuance and management of invoices per client.
\item \doxylink{struct_product}{Product} search with wildcards.
\item Global system statistics. 
\end{DoxyItemize}
